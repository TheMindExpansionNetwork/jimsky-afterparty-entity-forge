# Safety Boundaries

Closed by default:
- No public posting or outreach while unattended.
- No paid spend, subscriptions, domains, billing, wallets, or payment setup.
- No training/GPU/model downloads unless an awake operator approves exact scope.
- No secrets in repo, payloads, generated images, dataset captions, or HF metadata.
- No recursive cron creation from unattended jobs.

Allowed unattended:
- Repo docs, verifiers, manifests, generated synthetic logo concepts, private dataset packaging, local commits, safe private uploads when already authenticated.
